Política de Privacidade

Última atualização: 18 de junho de 2026

Esta Política de Privacidade descreve como o site Sergyo Eletrônico (sergyoeletronico.com) coleta, usa e protege as informações dos visitantes, em conformidade com a Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018).

1. Quem somos

Sergyo Eletrônico é um negócio individual (MEI) sediado em Goianinha/RN, Brasil, especializado em eletrônica automotiva para Fiat Uno. Contato: sergyoeletronic@gmail.com | (84) 99640-2534.

2. Dados que coletamos

Este site não possui formulários de cadastro ou área de login. Os dados coletados são limitados a:

  • Dados de navegação (Google Analytics): páginas visitadas, tempo de permanência, país/cidade de origem, tipo de dispositivo e navegador. Esses dados são anônimos e agregados — não identificam você pessoalmente.
  • Dados fornecidos voluntariamente: quando você clica no botão do WhatsApp e inicia uma conversa, qualquer informação que você compartilhar (nome, modelo do carro, dúvidas) é tratada diretamente via WhatsApp e não é armazenada neste site.

3. Como usamos os dados

  • Entender quais produtos geram mais interesse e melhorar o conteúdo do site
  • Medir o tráfego proveniente do YouTube, redes sociais e busca orgânica
  • Identificar problemas técnicos de navegação

Não usamos seus dados para publicidade direcionada, não vendemos dados a terceiros e não compartilhamos informações pessoais com ninguém.

4. Google Analytics

Utilizamos o Google Analytics 4 para análise de tráfego. Esse serviço usa cookies para coletar dados de navegação de forma anônima. O Google pode processar esses dados nos Estados Unidos de acordo com sua própria política de privacidade.

Para saber mais ou optar por não ser rastreado: tools.google.com/dlpage/gaoptout.

5. Cookies

Este site utiliza apenas os cookies técnicos do Google Analytics. Não usamos cookies de publicidade ou rastreamento de terceiros além desse. Você pode desativar cookies nas configurações do seu navegador sem perder nenhuma funcionalidade do site.

6. Links externos

Este site contém links para o WhatsApp, YouTube e Mercado Livre. Ao acessar esses serviços, você está sujeito às políticas de privacidade de cada plataforma, que são independentes desta.

7. Seus direitos (LGPD)

Conforme a LGPD, você tem direito a:

  • Confirmar se tratamos dados seus
  • Solicitar a exclusão de dados pessoais que eventualmente tenhamos
  • Revogar consentimento a qualquer momento
  • Obter informações sobre com quem compartilhamos dados (resposta: ninguém)

Para exercer qualquer desses direitos, entre em contato: sergyoeletronic@gmail.com.

8. Segurança

O site é hospedado no GitHub Pages com HTTPS obrigatório. Não armazenamos senhas, dados de cartão ou informações financeiras — todas as transações são processadas pela Kiwify ou combinadas diretamente via WhatsApp.

9. Alterações nesta política

Podemos atualizar esta política periodicamente. A data de última atualização sempre estará indicada no topo desta página. O uso continuado do site após alterações implica aceitação da versão atualizada.
